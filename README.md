# cn-chn-lab

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面，涵盖不同主题与用途。这些页面均为静态资源，可直接在浏览器中打开，无需额外依赖或服务端支持。仓库主要作为个人实验、演示或信息展示的集中存放处，不针对任何特定域名或网站。

## 目录结构

```
cn-chn-lab/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面所需的静态资源（图片、样式、脚本等）
└── README.md       # 本文件
```

- **pages/**：每个 HTML 文件为一个独立页面，文件名可反映其内容或功能。
- **assets/**：按需存放页面引用的资源文件，保持组织清晰。

## 页面归档说明

- 所有页面均为静态 HTML，兼容主流浏览器。
- 每个页面可能包含独立样式（CSS）与脚本（JavaScript），内嵌或引用自 assets 目录。
- 页面之间无关联，可单独访问或部署。
- 归档内容可能随时间更新或新增，不保证所有页面长期可用。
